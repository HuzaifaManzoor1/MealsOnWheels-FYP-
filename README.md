# 🍽️ Meals on Wheels

Meals on Wheels is a food delivery web application developed as a Final Year Project. It allows users to browse meals, place orders, and get food delivered easily.

## 🚀 Tech Stack

* Frontend: React.js
* Backend: Node.js (Express.js)
* Authentication & Database: Firebase
* Deployment: Vercel

## ✨ Features

* User Signup & Login (Firebase Authentication)
* Browse available meals
* Add to cart functionality
* Place orders online
* Real-time data handling using Firebase
* Responsive UI for better user experience

## 📁 Project Structure

Meals-on-Wheels/
│── client/        # React frontend
│── server/        # Node.js backend
│── README.md

## ⚙️ Installation & Setup

### 1. Clone the repository

git clone https://github.com/your-username/meals-on-wheels.git
cd meals-on-wheels

### 2. Install dependencies

# Frontend

cd client
npm install

# Backend

cd ../server
npm install

> Note: node_modules folder is not included in the repository. Run `npm install` to install all dependencies.

### 3. Run the project

# Start backend

npm run dev

# Start frontend

npm start

## 🔐 Environment Variables

Create a `.env` file in both client and server folders and add:

FIREBASE_API_KEY=your_key
FIREBASE_AUTH_DOMAIN=your_domain
FIREBASE_PROJECT_ID=your_project_id
